# OgeMath v7 (Qt Widgets + Qt Charts)

## Как открыть
1) Открой `OgeMath.pro` в Qt Creator.
2) Выбери комплект: **Desktop Qt 6.10.2 MinGW 64-bit** (или другой Qt6 с Charts).
3) Нажми **Run qmake**, затем **Rebuild**.

## Где лежат данные
Приложение ищет `data/*.json` рядом с проектом (а также пробует ../data, ../../data и т.п.).
Для демо здесь лежат пустые JSON. Замени их своими большими файлами.

## Важное
Если после правок появляются “сотни ошибок” — удали папку `build/` и пересобери с нуля.
